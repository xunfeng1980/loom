<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| Untrusted bytes -> container decoder | Offset/length arithmetic must reject malformed containers before slicing. |
| Feature flags -> decode permission | Unknown required features must fail closed before layout/table decode. |
| Container -> legacy payload codecs | Wrapped payload bytes must be extracted exactly and then handed to existing checked decoders. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-11-01 | Tampering | section directory | mitigate | Checked offset/length math and negative tests for truncated/overflow sections. |
| T-11-02 | Spoofing | payload kind | mitigate | Distinct magic values and payload-kind helper tests for `LMP1`/`LMT1`/`LMC1`. |
| T-11-03 | Denial of Service | section count | mitigate | Bound section count by input length and fixed entry size before allocation-sensitive parsing. |
| T-11-04 | Elevation of Privilege | unknown required feature | mitigate | Decode rejects unknown required bits before exposing wrapped payload. |

No high open threats remain if malformed containers fail before wrapped payload decode.
</threat_model>
