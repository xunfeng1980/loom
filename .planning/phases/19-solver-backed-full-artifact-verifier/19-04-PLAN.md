<objective>
Wire solver discharge reports into the artifact verifier so `ConstraintDischargeStatus::Discharged` is set only from successful Bitwuzla-backed evidence.
</objective>

<context>
After 19-03, Loom can emit SMT-LIB and run Bitwuzla. This plan connects the evidence to artifact facts without making structural verification depend on solver availability.
</context>

<threat_model>
- Threat: rejected/unsupported artifacts run solver or expose facts.
  Mitigation: solver integration only consumes accepted structural reports.
- Threat: skipped solver evidence still enables lowering.
  Mitigation: skipped reports preserve structural acceptance but do not set `Discharged` or lowering-ready trust.
- Threat: `sat` counterexamples are misclassified as unsupported.
  Mitigation: map counterexample evidence to fail-closed rejection diagnostics at the constraint-discharge stage.
</threat_model>
