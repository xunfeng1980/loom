## Decisions Made

- **Dev-time local helpers**: Placed emission and oracle helpers directly in each test file rather than in a shared module, to avoid `#[cfg(test)]` visibility issues when importing from library crates to integration test crates
- **Oracle strategy**: Vortex and Lance use `SourceOracleStrategy::SourceNativeScan` (true native scan); Parquet uses `SourceOracleStrategy::ArrowScan` (reads via Arrow reader)
- **LMA1 vs LMC2**: Legacy readability tests generate direct LMA1 bytes via `encode_arrow_semantic_payload`; other tests generate LMC2 via `encode_arrow_semantic_container_payload`
- **Legacy manifest assertions preserved**: Restored original function name assertions in `legacy_readability.rs` tests since MANIFEST.md files contain static references to the original API names
