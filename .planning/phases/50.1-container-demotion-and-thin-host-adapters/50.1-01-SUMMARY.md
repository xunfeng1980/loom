## Decisions Made
- Used Option<String> for tcb_status and artifact_role fields (not enum) — forward-compatible with future artifact kinds without breaking the public API
- LMC1 verification path intentionally untouched — LMC1 is structural-only and unaffected by LMC2/LMA1 demotion
- Deprecation via annotation rather than deletion — downstream ingress adapters (Parquet, Lance, Vortex) still use SourceIngressAcceptedArtifact for existing emission paths
