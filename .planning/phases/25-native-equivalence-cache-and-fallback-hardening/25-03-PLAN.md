<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| native buffers -> equivalence oracle | Native/JIT output is compared against interpreter/reference bytes before trust. |
| unsupported program -> fallback/fail-closed route | Unsupported accepted artifacts must not become native-success claims. |
| failed route -> later scan | Mismatch/cancel/error routes must not poison cache or emit partial rows. |
