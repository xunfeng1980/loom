# Phase 43.2-01 Summary: Deterministic Replay and Cache Drift Hardening

**Status:** Complete
**Date:** 2026-06-09
**Plan:** `43.2-01-PLAN.md`

## What Changed

- Added `NativeArrowSemanticCodegenReplayEvidence` in
  `loom_core::native_arrow_semantic`.
- Added `native_arrow_semantic_codegen_replay_evidence`, which emits positive
  replay evidence only for Phase 40 validated production native-codegen output.
- Added `validated_native_arrow_semantic_codegen_runtime_cache_key_with_shape`
  so production native-codegen cache identity can include projection,
  predicate, split, and policy inputs explicitly.
- Kept the existing cache-key helper stable by delegating it to the new
  shape-aware helper with `PredicateEnvelope::None` and a full-scan split.
- Added deterministic support/output/validation replay fingerprints:
  - artifact digest;
  - support fingerprint;
  - output buffer fingerprint;
  - reference/native trace fingerprints;
  - validation fingerprint;
  - runtime cache stable id and canonical input;
  - combined replay fingerprint.
- Included output-buffer fingerprinting in the production lowering fingerprint
  used by validated production native-codegen cache keys.
- Added `native_arrow_semantic_codegen_stability` tests for replay stability,
  backend/artifact drift, unsupported query shapes, unsupported support, and
  divergent output.

## Verification

Passed:

```sh
cargo test -p loom-core --test native_arrow_semantic_codegen_stability
cargo test -p loom-core --test native_arrow_semantic_codegen
cargo test -p loom-core --test native_arrow_semantic
```

## Evidence

- Repeated `LMC2(LMA1)` executions produce identical replay evidence.
- Repeated direct `LMA1` bridge executions produce identical replay evidence
  while preserving distinct direct-bridge artifact identity.
- Backend identity and artifact drift change replay/cache evidence.
- Projection, predicate, row-range split, and mismatched full-scan row counts fail
  closed before replay/cache admission until those semantics are implemented and
  validated.
- Unsupported and divergent executions cannot produce positive replay evidence.
