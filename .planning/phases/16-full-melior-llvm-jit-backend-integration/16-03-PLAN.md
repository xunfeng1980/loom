<objective>
Add MLIR verification and external-tool validation around the Phase 16 programmatic artifact, plus a skip-aware `scripts/melior-jit-test.sh` gate that reports optional evidence without breaking normal release verification on machines without compatible MLIR/LLVM.
</objective>

<context>
This plan validates the generated MLIR artifact and prepares the release gate shape. It does not require successful JIT execution yet; that is Plan 16-04.
</context>

<threat_model>
- Threat: optional toolchain absence gets mistaken for success.
  Mitigation: script output must distinguish `[PASS]`, `[SKIP]`, and `[FAIL]`, and strict mode must convert skip to fail.
- Threat: incompatible local LLVM version silently accepts partial evidence.
  Mitigation: compare toolchain facts against `EXPECTED_MLIR_MAJOR` and surface `toolchain-version-mismatch`.
- Threat: `scripts/mvp0-verify.sh` becomes unusable on machines without MLIR/LLVM.
  Mitigation: wire Phase 16 gate as optional skip-aware evidence unless strict env var is set.
</threat_model>
