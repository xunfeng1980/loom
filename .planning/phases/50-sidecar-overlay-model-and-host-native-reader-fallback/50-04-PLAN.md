| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-50-11 | Tampering | loom sidecar embed --ir file | mitigate | IR file is decoded and verified via decode_l2core_program / verify_l2_core_bytes before embedding; invalid IR is rejected with diagnostic |
| T-50-12 | Denial of Service | CLI embedding large files | accept | Hash computation over column chunks is O(n); large files may be slow; this is a CLI tool, not a latency-critical path |
