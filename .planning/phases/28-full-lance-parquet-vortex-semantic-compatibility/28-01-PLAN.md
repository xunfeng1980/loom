<objective>
Create the Phase 28 semantic compatibility matrix contract.

Purpose: make Vortex compatibility claims machine-readable and impossible to
overclaim before widening implementation coverage.
Output: row/report types, mapping from `VortexEncodingCoverage`, and focused
tests for accepted, unsupported, canonicalized, and native-supported states.
</objective>

<threat_model>
T-28-01: False compatibility claims. A canonicalized or fallback-supported row
could be reported as structured or native support. Mitigation: typed enums and
tests must reject impossible row combinations.
</threat_model>
