# 42-02 Plan: Lance/Parquet Schema Coverage and Native Disposition

## Objective

Record Lance/Parquet schema coverage using the existing `LMC2(LMA1)` source
semantic handoff and make native disposition explicit per schema family.

## Tasks

1. Add Parquet schema matrix tests.
   - Cover nullable fixed-width primitives, Boolean, Utf8, Struct, and List
     using existing Arrow semantic roundtrip helpers.
   - Assert default emission is `LMC2(LMA1)` and verifier accepted.
   - Assert native-supported only for Phase 35 fixed-width primitive subsets;
     Utf8/List/Struct rows are interpreter-only.

2. Add Lance schema matrix tests.
   - Mirror the Parquet schema rows using Lance dataset writer/oracle helpers.
   - Preserve direct `LMA1` as bridge regression evidence only.

3. Update `42-COVERAGE-MATRIX.md`.
   - Add Lance/Parquet rows with source oracle, verifier, native, and fallback
     disposition.
   - Record unsupported/deferred shapes separately from accepted interpreter-only
     rows.

## Acceptance Criteria

- `cargo test -p loom-parquet-ingress --test phase42_source_schema_matrix`
  passes.
- `cargo test -p loom-lance-ingress --test phase42_source_schema_matrix`
  passes.
- `bash scripts/full-arrow-semantic-compatibility-test.sh` still passes.

## Non-Claims

Accepted source semantic emission is not source-data correctness, source-format
semantic proof, DuckDB native consumption, or ABI freeze.
