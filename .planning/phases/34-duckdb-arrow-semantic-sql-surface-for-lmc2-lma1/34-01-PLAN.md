<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| untrusted artifact bytes -> internal handle | Local files may be malformed `LMC2`, direct `LMA1`, or unrelated bytes. |
| Rust Arrow arrays -> C++ release ownership | Exported Arrow C Data objects cross language ownership. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-34-01-01 | Tampering | artifact bytes | mitigate | Verify/unwrap/decode in Rust before exposing schema facts. |
| T-34-01-02 | Denial of Service | batch/column shape | mitigate | Reject unsupported multi-batch and unsupported types before DuckDB scan state creation. |
| T-34-01-03 | Elevation of Privilege | public ABI | mitigate | Keep new symbols in internal header and cbindgen exclusions only. |
| T-34-01-04 | Repudiation | Arrow C ownership | mitigate | Tests release exported arrays/schemas and check no double-release path is required. |
</threat_model>
