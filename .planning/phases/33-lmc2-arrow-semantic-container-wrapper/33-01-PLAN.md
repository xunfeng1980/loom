<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| untrusted bytes -> LMC2 codec | Local artifact bytes may be malformed, truncated, or feature-incompatible. |
| LMC2 wrapper -> inner LMA1 | The wrapper must not allow malformed or missing inner semantic payload bytes to be treated as accepted. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-33-01-01 | Tampering | LMC2 header and section directory | mitigate | Checked version, header length, section count, offset, and length validation reject malformed bytes. |
| T-33-01-02 | Spoofing | required feature bits | mitigate | Unknown required feature bits fail closed before inner payload acceptance. |
| T-33-01-03 | Denial of Service | length arithmetic | mitigate | Use checked arithmetic and bounded reads for all lengths and offsets. |
</threat_model>
