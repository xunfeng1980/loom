|----------|-------------|
| Prepared native buffer to DuckDB vector | Raw native bytes cross into host vector memory. |
| Route decision to scan output | A stale or failed route must not emit rows. |
| Interpreter Arrow state to scan output | Arrow arrays remain release-callback owned until scan state teardown. |
